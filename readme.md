# F5 AS3 Declarations Archive

This repository contains my collection of **F5 BIG-IP AS3 (Application Services 3)** declarations.  
It serves as a personal archive and reference for automating BIG-IP configurations using **Declarative Onboarding (DO)** and **AS3** templates.
